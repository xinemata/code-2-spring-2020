Description

The look of a word has an enormous impact on how we understand the word. For centuries, typographers have explored and developed different ways to make type more expressive. Living in an era where computer programming has become increasingly accessible to designers, typography has adopted the principles of modular and parameterized design and became more interactive.

For Project #2, you will pick three different words to create three interactive posters. Incorporate textToPoints() and / or classes and objects in the sketches. Your choice of shape, color, motion, and interaction should thoughtfully reflect on the meaning of the word.

textToPoints() Basics

textToPoints() is a function in p5 that converts text into points on the screen. It’s an exciting feature that will allow us to manipulate type in new and surprising ways. In order for textToPoints() to work, you have to use a specific font format called OpenType font (OTF) in your sketches. Here are a couple direct links for you to download OTF fonts:

  • Copenhagen Grotesk Nova
  • Myriad Pro
  • Letter Gothic

Loading Font

Once you download the files to your computer, open the p5 web editor. Go to Sketch > Add File and drag an OTF file to the dialogue box. Once it’s uploaded you should see the file showing up in your sketch folder on the left side of the editor.

Now to load the font file, you need to create a global variable to hold the font and also load the font file from the preload() function. Comparable to setup() and draw(), the preload() function makes sure that your font is loaded before the browser loads the rest of the website.

The preload() function should be placed before the setup() function. And inside setup(), you should load the grotesk variable inside textFont(). This tells the program that you will be using Grotesk as your default font.

FAQ

Q: What is polar coordinate system?

We have been using the Cartesian coordinate system in p5.js, where every position is described by its x and y values on the plane. Contrary to the Cartesian coordinate system, the polar coordinate system describes the position using (1) distance from the center and (2) degrees of angle. The syntax for the polar coordinate system is (distance from the center, degrees of angle).
